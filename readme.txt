comandos mas usados

* git add .  = agregar mis cambios al area de seguimiento

* git status = ver el estado de los cambios en nuestros archivos

* git commit - m " aqui va un mensaje " = confirmacion de los archivos a los que se les ha hecho seguimiento 

* git push origin nombre-de-la-rama = subir nuestros cambios a nuestra rama remota, nube.
